# PrimeScale

Remote tech staffing platform by People Prime Worldwide. Companies post roles on PrimeScale and receive vetted candidates within 24 hours.

## Getting started

```bash
cd ~/Projects/PrimeScale
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project structure

- `src/app/` — pages and API routes
- `src/components/` — UI components
- `public/` — images and static assets
- `data/roles.json` — role submissions (local dev only)

## Deploy

Deploy to [Vercel](https://vercel.com) for a live 24/7 website. Connect your GitHub repo and Vercel will handle builds automatically.
